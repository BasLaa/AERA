Detailed Description

TemplateTimingsUpdater is a helper class for retrieve_imdl_fwd, etc. to save an f_imdl template timings and temporarily set them from another f_imdl. We won't need this if we implement https://github.com/IIIM-IS/replicode/issues/137

Create a TemplateTimingsUpdater and save the timings from f_imdl, then set them to the values from other_f_imdl so that Match will ignore any difference. The destructor will restore the values in f_imdl.
